## Description

Using the CIS-CAT Pro results from **Lab 5**, produce a structured gap analysis report. For **every failing control** in your results document:

1. Control ID and title
2. CIS Level (1 or 2)
3. Current state description (what was found)
4. Target state and hardened configuration
5. Remediation command or configuration change
6. Estimated effort (Low / Medium / High)
7. Risk if not remediated

---
