## Description

Submit the current state of your Capstone Project automation for **one server** in your infrastructure. Must include:

1. Minimum **15 hardening controls** implemented
2. Variables externalized in a separate file
3. `README.md` documenting: purpose, prerequisites, variable descriptions, and how to run
4. Summary of current CIS-CAT or STIG scan score for the target server

---

## Deliverable(s)

Submit a compressed `.zip` of your playbook/config directory **+** `README.md` **+** CIS-CAT or STIG scan report (PDF or HTML).

| Criterion | Points |
