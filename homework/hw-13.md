## Description

Design a **Privileged Access Management (PAM) policy** for a 3-tier web application environment (web server, app server, database server). The policy must define:

1. **Role definitions matrix** (Developer, DBA, SysAdmin, Security Admin, Read-Only Auditor)
2. **SSH access matrix** (who can SSH to which server and how)
3. **MFA requirements** by role
4. **Session recording policy** (when and how sessions are recorded)
5. **Password/key rotation schedule**
6. **Emergency break-glass procedure** with audit trail requirements

---

## Deliverable(s)

Submit a professional policy document (2–3 pages) using the provided template **plus** a visual access matrix diagram.
