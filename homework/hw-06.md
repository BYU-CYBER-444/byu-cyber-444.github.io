## Description

Using the provided Windows Server 2022 STIG checklist template (STIG_Checklist_HW6.xlsx on LMS), manually evaluate and document **15 specific STIG items** assigned by your instructor. For each item:

1. STIG Rule ID and title
2. Status: Open / Not a Finding / Not Applicable / Not Reviewed
3. Finding Details — the exact configuration observed
4. Comments explaining your status determination
5. Compensating control (if status is Not Applicable)

---

## Deliverable(s)

Submit the completed checklist template `.xlsx` with all 15 items fully documented.
