## Description

Using your findings from **Lab 14** (the compromised Ubuntu server exercise), complete a formal post-incident report using the NIST SP 800-61 structure. Include:

1. **Executive Summary** (1 paragraph, non-technical audience)
2. **Incident Timeline** (chronological table of events with timestamps)
3. **Root Cause Analysis** (what vulnerability or misconfiguration was exploited)
4. **Containment and Recovery Actions** (what you did and why)
5. **Lessons Learned** (what would you do differently)
6. **5 Hardening Recommendations** to prevent recurrence
