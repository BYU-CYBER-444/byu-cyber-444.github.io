## Description

Draft a professional **2-page Patch Management Policy** for a fictional 200-seat organization ("Acme Financial Corp"). The policy must include:

1. Scope and applicability
2. Patch classification table (Critical / High / Medium / Low) with SLA timelines
3. Testing and staging process (dev → staging → production)
4. Emergency patching procedure (out-of-band)
5. Rollback and exception process
6. Roles and responsibilities (SysAdmin, CISO, Change Control Board)
