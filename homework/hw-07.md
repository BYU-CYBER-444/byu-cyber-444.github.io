## Description

Create a detailed comparison table documenting **10 Windows hardening controls** applied in Lab 7. For each control record:

1. CIS Benchmark control ID and title
2. DISA STIG rule ID (if applicable)
3. Default state in a fresh Windows Server 2022 install
4. Hardened state
5. Implementation method (GPO path / Registry key / PowerShell command)
6. Security impact (CIA triad analysis)
7. Potential operational impact

---

## Deliverable(s)

Submit an Excel table **plus** a one-paragraph executive summary identifying the 3 controls with the highest security ROI.
