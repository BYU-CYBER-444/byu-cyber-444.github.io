## Description

Extend the Ansible playbook from **Lab 9** with the following additions:

1. A **handler** that restarts the `auditd` service whenever audit rules change
2. A separate `vars/main.yml` file externalizing all site-specific settings (org name, log retention days, allowed SSH ciphers)
3. A `pre_task` that checks the target OS is Ubuntu 20.04 or 22.04 and **fails gracefully** with a clear error message if not
4. At least **5 additional CIS Level 2 controls** not covered in Lab 9

---

## Deliverable(s)

Submit your Ansible playbook directory (compressed `.zip` or Git repo link) **+** playbook execution output showing a successful idempotent run (0 changed on second run).
