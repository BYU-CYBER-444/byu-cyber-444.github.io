## Description

For the provided Docker Compose application stack (`DockerCompose_HW12.yml` — available on LMS), document **10 security improvements** using CIS Docker Benchmark controls. For each:

1. CIS Docker Benchmark control number and title
2. Current state in the provided Compose file
3. Risk description
4. Remediation: modified Docker Compose YAML snippet
5. Priority (Critical / High / Medium)
6. Verification command

---

## Deliverable(s)

Submit a completed checklist table (Excel or structured document) **plus** a modified `docker-compose.yml` with all 10 improvements applied.
