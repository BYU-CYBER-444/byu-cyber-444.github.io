## Description

Create a PowerShell script that automates stale account remediation in Active Directory:

1. Queries AD for all user accounts with `LastLogonDate` > 90 days ago
2. Exports results to a CSV with columns: `SamAccountName`, `LastLogonDate`, `OU`, `Email`
3. Disables each account and sets `Description` to `Disabled by automated policy [DATE]`
4. Outputs a summary (simulated with `Write-Host`) to the IT admin

---

## Deliverable(s)

Submit your `.ps1` script **+** CSV export showing at least 3 test accounts **+** screenshot of the accounts disabled in ADUC.
