## Description

Analyze the provided **500-line `auditd` log excerpt** (`AuditLog_Exercise.txt` — available on LMS). Identify and document:

1. At least **3 suspicious events** with full details (timestamp, user, UID, command, result)
2. **MITRE ATT&CK technique ID and name** for each event
3. A chronological **attack timeline narrative**
4. The attacker's apparent **objective** based on log evidence
5. **Three hardening recommendations** to prevent each identified technique

---

## Deliverable(s)

Submit a written report (1–2 pages) with an evidence table (referencing specific log lines), MITRE mappings, timeline narrative, and recommendations.
