## Course Description

This course provides advanced, hands-on instruction in enterprise-grade operating system hardening,
configuration management, and infrastructure automation. Students master CIS Benchmark and DISA STIG
compliance for Linux and Windows Server environments, implement infrastructure-as-code security using
Ansible and Terraform, and develop automated patch management pipelines.

Topics include identity and access management for administrators, centralized logging and audit trail
architecture, virtualization and container security hardening, and the systems administrator's role in
incident response. The course culminates in a Final Project where students design, harden,
document, and defend a fully compliant server infrastructure meeting CIS Level 2 and applicable DISA
STIG requirements.
